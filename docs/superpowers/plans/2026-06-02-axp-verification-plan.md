# AXPlain.ai 기능 검증 계획 (Verification Plan)

> **실행 방법:** Chrome DevTools MCP로 https://axp-patent-app.vercel.app/ 에 접속하여
> 각 체크리스트를 직접 조작·확인한다. 목업 데이터임을 감안하여 **UI 기능 요소** 중심으로 점검한다.

## 검증 기준

- ✅ **통과**: 기능이 의도한 대로 동작함
- ⚠️ **부분**: 동작하나 UX 개선 필요 또는 mock 데이터로 내용만 다름
- ❌ **실패**: 클릭해도 반응 없음, 오류, 흐름 단절
- 🔲 **미구현**: 의도적 미구현 (toast 표시 또는 alert)
